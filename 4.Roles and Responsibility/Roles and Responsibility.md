# Nutrition Assistant - Roles and Responsibilities

## Overview

The Nutrition Assistant is a MERN Stack based web application that helps users maintain a healthy lifestyle through meal planning, calorie tracking, nutrition analysis, and personalized diet recommendations. The system defines clear roles and responsibilities to ensure smooth operation and efficient management of user data and nutrition services.

---

# 1. User Responsibilities

The User is the primary actor who interacts with the application.

### Responsibilities

- Register a new account with accurate information.
- Log in securely using registered credentials.
- Maintain personal profile information.
- Update age, weight, height, and fitness goals.
- Search and select food items.
- Record daily meals.
- Monitor calorie intake.
- View nutrition reports.
- Follow personalized diet recommendations.
- Track health progress regularly.
- Maintain accurate nutrition logs.

---

# 2. Administrator Responsibilities

The Administrator manages the overall application and database.

### Responsibilities

- Manage user accounts.
- Monitor system performance.
- Maintain the food database.
- Add new food items.
- Update nutrition information.
- Remove invalid data.
- Manage user feedback.
- Generate system reports.
- Monitor security.
- Perform database backups.

1. User registers an account.
2. User logs into the application.
3. User updates personal profile.
